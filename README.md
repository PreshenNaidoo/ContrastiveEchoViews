# Spatiotemporal Contrastive Learning for Echocardiography View Classification

The complete source code will be released with the research paper.

## Packages

The following packages are required:

- **Python** 3.10  
- **TensorFlow** 2.15.0  
- **Keras** 2.15.0  
- **Pandas** 1.5.3  
- **NumPy** 1.24.4  
- **OpenCV** 4.5.5.64
